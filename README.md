# Calculus Plot and LaTeX Example

This repository demonstrates an automated GitHub Actions workflow.

The `calculus` folder contains:

- `plot.py`, which plots \(y=x^2\) for \(-2 \leq x \leq 2\)
- `main.tex`, which includes the formula \(x^2\) and the generated plot
- `f1.png`, automatically generated by the Python program
- `main.pdf`, automatically compiled from the LaTeX source

Whenever a commit is pushed to the `main` branch, GitHub Actions:

1. Runs the Python program.
2. Replaces `calculus/f1.png`.
3. Compiles the LaTeX document.
4. Replaces `calculus/main.pdf`.
5. Commits the generated files back to the repository.
